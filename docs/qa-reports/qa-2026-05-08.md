## 권장 조치

### 🔴 오류 수정 (필수)

**`PreviewPanel.tsx:15` — useEffect 내 setState 제거**

`window.location.origin`은 렌더링 중에 변하지 않으므로, `useState` 초기값으로 직접 설정하는 것이 올바른 방법입니다:

```tsx
// ❌ 현재 코드
const [previewOrigin, setPreviewOrigin] = useState('');
useEffect(() => {
  setPreviewOrigin(window.location.origin);
}, []);

// ✅ 수정 코드 (클라이언트 전용이므로 초기값으로 설정)
const [previewOrigin] = useState(() =>
  typeof window !== 'undefined' ? window.location.origin : ''
);
```
