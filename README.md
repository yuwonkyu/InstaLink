# LinkTree

1인 사업자·프리랜서를 위한 링크 페이지 서비스입니다.  
PT 트레이너, 필라테스 강사, 미용실, 프리랜서 등 누구든 링크 하나로 자신을 소개하고 고객 상담까지 연결할 수 있습니다.

## 주요 기능

- 개인 프로필 페이지 (`/username`)
- 서비스·가격 안내
- 카카오톡 상담 연결 CTA
- 인스타그램 링크 연결
- 모바일 우선 반응형 UI

## 기술 스택

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- Vercel
