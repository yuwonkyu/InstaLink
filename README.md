# SaveMe PT Link Page

인스타그램 프로필 링크에 연결하는 모바일 우선 PT 예약용 링크트리 웹페이지입니다.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- npm
- Vercel

## Structure

app/
[username]/page.tsx
globals.css
layout.tsx
page.tsx
components/
Footer.tsx
Menu.tsx
Profile.tsx
Reservation.tsx
data/
mockData.ts

## Run

```bash
npm install
npm run dev
```

기본 샘플 페이지는 /coach-jiyun 으로 연결됩니다.

## Deploy

Vercel에 Git 저장소를 연결한 뒤 기본 설정으로 배포하면 됩니다.
